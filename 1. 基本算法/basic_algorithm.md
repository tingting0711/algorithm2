# 基础算法
## 位运算

- 基本操作
  
  - 与 x & y
  - 或 x | y
  - 非 !x
  - 异或 x ^ y(不进位加法)
  - 补码 - x = ~x + 1(负x = 取反x + 1)
  
- 移位运算

  - 左移 1 << n = 2 ^ n
  - 右移 n >> x = n / 2^x (补符号位)

- 配偶：正反向边

  - 0  ^ 1 = 1, 1 ^ 1 = 0 (0和1为配偶)

- lowbit运算

  - 快速求出整数N的最后一位1是什么
  - lowbit(1110001001000) = 1000
  - (~n + 1) & n = (-n) & n
