- 基本操作

  - &， /， !，^

  - \> >, \<<

- 补码

  - 负数
  - 取反加一

- 用 **异或1** 实现配偶

  - e[inedx], e[index ^ 1]; (0,1)(2,3)(4,5)(...)
  
- lowbit 运算
  
- 最低位为1
  - 快速求出数有多少个1

> lowbit

```
int lowbit(int n)
{
		return (~n + 1) & n; // return (-n) & n;
}
```
